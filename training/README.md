# Depression Model Training (AU-based)

This folder contains a minimal, reproducible pipeline to train a depression-related model
from AU features exported by the FacePsy scan UI.

## What you need
1. Exported scan JSON files (from the app's "Export JSON" button).
2. Labels for each scan (e.g. PHQ-9 score).

## Synthetic data (for testing only)
You can generate a synthetic dataset to test the training pipeline.
This is NOT real data and should NOT be used to claim accuracy.

```
python generate_synthetic.py \
  --exports ./data/exports \
  --out ./data/dataset/dataset.csv \
  --labels ./data/labels/labels.csv \
  --count 200
```

## Recommended folder layout
```

```

## labels.csv format
Minimum columns:
```
sessionId,phq9_score
session_1719999999999_abcd1234,12
session_1719999999999_efgh5678,3
```

Optional columns:
```
userId           # recommended for group-split by user
scanTimestamp    # can be used to match if sessionId missing
```

## Step 1: Build dataset
```
python build_dataset.py \
  --exports ./data/exports \
  --labels ./data/labels.csv \
  --out ./data/dataset.csv
```

## Step 2: Train model
Classification (PHQ-9 >= 5):
```
python train_model.py \
  --data ./data/dataset.csv \
  --task classification \
  --label-col phq9_score \
  --out-model ./data/depression_model.joblib \
  --out-meta ./data/depression_model_meta.json
```

Regression (predict PHQ-9 0-27):
```
python train_model.py \
  --data ./data/dataset.csv \
  --task regression \
  --label-col phq9_score \
  --out-model ./data/depression_model.joblib \
  --out-meta ./data/depression_model_meta.json
```

## Notes
- This pipeline uses AU/pose/eye features extracted from exported scans.
- Reliable models need many participants and repeated labels (PHQ-9) over time.
- This is for research and prototyping only, not medical diagnosis.
